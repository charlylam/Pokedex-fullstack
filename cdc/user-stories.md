# User Stories – Projet Pokédex

| Version | En tant que | Je souhaite pouvoir | Afin de |
|--------|-------------|--------------------|--------|
| Version minimale | visiteur | consulter la page des équipes | voir la liste des équipes existantes |
| Version minimale | visiteur | consulter la page / modale d'une équipe | consulter ses caractéristiques |
| Version minimale | visiteur | créer une équipe | l'administrer |
| Version minimale | visiteur | modifier le nom d'une équipe | l'administrer |
| Version minimale | visiteur | ajouter un Pokémon à une équipe | l'administrer |
| Version minimale | visiteur | retirer un Pokémon d'une équipe | l'administrer |
| Version minimale | visiteur | supprimer une équipe | l'administrer |
| Version attendue | visiteur | consulter la page des Pokémons | voir la liste des Pokémons existants |
| Version attendue | visiteur | consulter la page / modale d'un Pokémon | consulter ses caractéristiques |
| Version attendue | visiteur | consulter la page récapitulative des types | lister les différents types existants |
| Version attendue | visiteur | consulter la page récapitulative d'un type | lister les Pokémons de ce type |
| Version attendue | visiteur | accéder à une page d'inscription | me créer un compte |
| Version attendue | visiteur | accéder à une page de connexion | me connecter et profiter des droits des membres |
| Bonus | visiteur | comparer deux Pokémons | m'aider à faire mon choix |
| Bonus | visiteur | rechercher un Pokémon par son nom | le retrouver facilement via une barre de recherche |
| Bonus | visiteur | voter pour un Pokémon | montrer mon intérêt pour ce Pokémon |
| Bonus | visiteur | voir le nombre de votes d'un Pokémon | voir l'intérêt général de ce Pokémon |
| Bonus | visiteur | consulter le podium des Pokémons | voir les 10 Pokémons les plus populaires |
